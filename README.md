# YelpCamp
YelpCamp is a platform for viewing and booking beautiful campsites located in all parts of the world. 

## Demo
![Yelp Camp Demo](yelpcamp_demo.gif)

# Technologies 

## Frontend 
- Bootstrap
- EJS

## Backend
- Express
- MongoDB

# Installation  
Follow the instructions below to set up the environment and run this project on your local machine. 

1. Clone this repository 
```bash
 git clone https://github.com/YazanJ/YelpCamp.git
```

2. Enter the following commands in terminal 
```bash
npm install
node app.js
```
